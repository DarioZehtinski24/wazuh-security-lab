# Wazuh Security Lab

A hands-on cybersecurity lab built using:

- Ubuntu Server 24.04
- Docker
- Wazuh SIEM/XDR
- VMware Workstation

This project documents the process of deploying, configuring, testing, and improving a Wazuh-based security monitoring environment.

---

# Objectives

- Learn SIEM fundamentals
- Deploy Wazuh using Docker
- Configure Wazuh agents
- Test detection capabilities
- Explore MITRE ATT&CK mappings
- Perform threat hunting
- Improve system hardening and security posture
- Document findings and troubleshooting

---

# Current Features

- Wazuh single-node deployment
- Ubuntu Server monitoring agent
- SSH authentication monitoring
- Security Configuration Assessment (SCA)
- Threat hunting via Events
- Custom anomaly detection experiments
- Docker-based deployment
